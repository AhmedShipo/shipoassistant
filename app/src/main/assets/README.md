# Shipo Assistant - مساعدك الذكي على أندرويد

مساعد شخصي متعدد الوظائف يعمل على أجهزة أندرويد، مبني باستخدام تقنيات الويب (HTML, CSS, JavaScript) ومحزّم كتطبيق أندرويد باستخدام Apache Cordova (عبر بيئة AIDE). يهدف هذا التطبيق إلى توفير تجربة مساعدة ذكية ومتكاملة للمستخدمين.

---

## 🌟 الميزات الرئيسية

* **نظام دردشة ذكي:** واجهة محادثة تفاعلية للرد على استفسارات المستخدمين.
* **الطقس الفوري عبر GPS:**
    * يعرض بيانات الطقس الحالية، توقعات دقيقة (دقيقة بدقيقة لمدة ساعة)، توقعات ساعة بساعة (لمدة 48 ساعة)، وتوقعات يومية (لمدة 8 أيام).
    * يستخدم الموقع الجغرافي (GPS) الخاص بجهازك للحصول على أحدث بيانات الطقس من OpenWeather One Call API 3.0.
    * زر "تحديث الطقس" خاص لجلب البيانات عند الطلب.
* **بحث ويكيبيديا:**
    * استبدال البحث التقليدي بوظيفة بحث في ويكيبيديا.
    * يعرض ملخصاً لنتائج البحث مع روابط للمقالات الكاملة.
    * إمكانية "حفظ كمرجع" لنتائج البحث (تخزين مؤقت في Local Storage).
* **عرض الخرائط (Google Maps):**
    * دمج شاشة لعرض الخرائط التفاعلية (يتطلب مفتاح Google Maps API وتفعيل الفوترة من جوجل).
* **دعم المرفقات (قيد التطوير):** وظيفة إرسال الملفات (جاهزة لاستقبال صور من الكاميرا/المعرض، وملفات).
* **تصميم مستجيب:** واجهة مستخدم بسيطة وجذابة تتكيف مع أحجام الشاشات المختلفة.

---

## 🛠️ التقنيات المستخدمة

* **الواجهة الأمامية (Frontend):** HTML5, CSS3, JavaScript (ES6+)
* **بيئة تطوير وتجميع التطبيقات (Mobile App Framework):** Apache Cordova (عبر AIDE)
* **واجهات برمجة التطبيقات (APIs):**
    * **OpenWeatherMap API:** لبيانات الطقس (One Call API 3.0).
    * **Google Maps Platform API:** لخدمات الخرائط.
    * **MediaWiki API (Wikipedia):** لخدمات البحث في ويكيبيديا.

## 🚀 كيفية الإعداد والتشغيل

1.  **الاستنساخ/الحصول على الكود:**
    * تأكد من أنك قمت بنسخ أو نقل جميع ملفات المشروع إلى بيئة AIDE على جهاز أندرويد الخاص بك.

2.  **الحصول على مفاتيح API:**
    * **OpenWeatherMap API Key:**
        * قم بزيارة [OpenWeatherMap](https://openweathermap.org/) وقم بإنشاء حساب.
        * بعد تسجيل الدخول، اذهب إلى قسم "API keys" (مفاتيح API).
        * **الأهم:** تأكد من أن المفتاح الخاص بك تم تفعيله لاشتراك **"One Call by Call"** (حتى لو كانت الخطة المجانية، يجب تفعيلها).
        * استبدل `'YOUR_OPENWEATHER_API_KEY'` في ملف `js/assistant_data/connectivity/externalCapabilities.js` بمفتاحك الخاص.
    * **Google Maps Platform API Key:**
        * قم بزيارة [Google Cloud Console](https://console.cloud.google.com/) وقم بإنشاء مشروع جديد (إذا لم يكن لديك).
        * **هام:** يتطلب Google Maps Platform تفعيل "حساب فوترة" (Billing Account) حتى للخطة المجانية (عادةً ما تحصل على رصيد مجاني شهري كبير).
        * قم بتمكين APIs التالية لمشروعك: `Maps JavaScript API`, `Geocoding API`.
        * انتقل إلى "APIs & Services" > "Credentials" وقم بإنشاء مفتاح API جديد.
        * **لزيادة الأمان:** قم بتقييد مفتاح API بحيث يعمل فقط مع واجهات برمجة التطبيقات التي تحتاجها ومنطاقك (HTTP referrers) المناسبة، مثل `file://*` للاختبار المحلي أو `http://localhost:*`.
        * استبدل `YOUR_Maps_API_KEY` في ملف `index.html` بمفتاحك الخاص (داخل وسم `<script>`).

3.  **الأذونات المطلوبة (Android):**
    * يحتاج التطبيق إلى إذن الوصول إلى **الموقع الجغرافي الدقيق (GPS)** لجلب بيانات الطقس. عند تشغيل التطبيق لأول مرة، قد يطلب منك نظام أندرويد منح هذا الإذن. تأكد من الموافقة عليه.
    * في بعض الحالات، قد تحتاج إلى إضافة هذا الإذن يدوياً في ملف `AndroidManifest.xml` الخاص بمشروع Cordova الخاص بك:
        ```xml
        <uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
        ```

4.  **تشغيل التطبيق في AIDE:**
    * افتح مشروعك في AIDE.
    * قم ببناء (Build) وتشغيل (Run) التطبيق على جهازك أو المحاكي.
